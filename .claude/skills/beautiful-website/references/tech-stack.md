# Preferred Tech Stack

## For landing pages and simple sites:
- Single HTML file with inline CSS/JS
- Tailwind CSS via CDN (https://cdn.tailwindcss.com)
- Google Fonts
- CSS animations (no heavy JS libraries unless needed)
- Deploy to Vercel

## For multi-page sites:
- Next.js 14+ with App Router
- Tailwind CSS
- Vercel for deployment
- Supabase if backend/database needed

## For web apps:
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase for backend
- Vercel for deployment

## Always:
- Mobile-first responsive design
- Google Fonts (never system fonts for display text)
- CSS custom properties for theming/color palette
- Semantic HTML
- Proper meta tags and Open Graph tags
